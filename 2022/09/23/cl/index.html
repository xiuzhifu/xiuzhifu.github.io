CL语言

Posted on 2022-09-23

需求

我一直想做一个游戏，主要给计算机系的朋友玩的游戏，里面有各种机器人，需要通过编程控制他们，所以，一直想找一个足够简单，容易使用的语言，因为只会作为脚本使用，所以，并不需要有多强大。Io，TCL两门语言，都是比较符合我的选择的

Io

Io是一个完全面向对象的语言，所有的东西都是对象，操作就是发送消息，概念很统一。

Tcl

Tcl是Tool Command Language的缩写，如其名，写代码就是在写各种命令:

            set A "hello world"
            if {true} {
                print A
            }

如上代码，一行是一个命令，以换行符作为结束，第一个元素是命令的操作，要做什么，后面的元素都是参数，如果一个参数，是由多个命令构成的,不能写在一行，需要用大括号括起来来跨行。这样的设计使得Tcl的模式非常的简单，只有两种结构，一个单行命令，一个跨行命令。

定义

实现一门脚本语言，不考虑效率，其实是很简单的，而Tcl语言的语法也没有，就两种结构，并不需要识别具体的if，while是什么的，留给虚拟器实现就行了，Parser非常好写，其实，这样其实跟Lisp是很像的，可以做到用相同的概念，做不同的事情。举个例子:

            def person {
                name: "xx"
                age: 12
                sex = 1
            }

我想给用Lua写的游戏添加一个配置，又不想把Lua当成配置，就可以用上面的定义来写配置，在lua里面生成一个对象来使用，例如:

            local A = require "A.vm"
            A.loadstring "
            def person {
                name: "xx"
                age: 12
                sex = 1
            }
            "
            local persion = A.person
            print(persion.name)

我现在又有一个需要，需要把Persion转换成XML结构，也可以:

            local V = require "B.vm"
            B.loadstring "
            def person {
                name: "xx"
                age: 12
                sex = 1
            }
            "
            print(B.toxml())

这样的话，不管语言怎么变，Parser是不需要变化的，只需要根据需求实现对应虚拟机的功能就可以了。

不同

  1. 上面的定义中，我用了name: “xx” set=1这样的写法，其实”:”, “=”才是命令，我为了形式上更直觉一点，方便识别跟书写，设计了一个语法糖，命令可以跟第一个参数互换:

             = name "mama"
         name = "mama"

上面两个命令是等价的，当第一个跟第一个元素都不是命令的时候，才识别错误。

  1. 在就是再Tcl中，要取一个变量的值，需要添加$符号,这是因为,如果没有这个概念的话，就不能区分需要的是变量本身，还是变量的值，例如:

             set name "who"
         def A {
             name: "who"
             who: "I"
         }
         print A.name//who
         print A.$name//I

因为这样看起来是用了多一个符号来实现的，所以，我的做法是再多一个符号:)

            print A.name//I
            print A:name//who

我用冒号来代表错做本身，点操作需要取变量的值，这样的话，我是觉得没那么突兀，暂时是这样考虑的，后面也许发现不好用我会改回Tcl的设计

  1. {} 跟(),[]是等价的,也是一个语法糖，if {true} {print “hello world” }可以写成 if (true) {print “hello world”}

完整的命令

  1. set A B 把B的值赋值给A，A必须是变量,例如:

             set x 1  set x "a" set y x

  2. set A.i B 把B的值赋值给对象A的键i

  3. def Object {
    key:value
    key=value
    } 定义对象

  4. = A B; : A B 单独使用的使用，跟set相同，作为def的子命令的时候，会设置定义对象的键值对。

  5. puts A.i B puts A:i B 把B的值赋值给对象A的键i。

  6. if {true} {body}

  7. while {true} {body}

  8. for start end step {body}

  9. print A print A.i print A:i 打印变量A的值

后续

作为一门简单的脚本语言，我是没打算提供的函数，因为功能不会很复杂，多了这这个反而多了一个概念，后面可以增加支持多源文件，如果功能复杂，可以用多个文件来组织代码就可以了。
